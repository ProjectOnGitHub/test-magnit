**Тестовое задание Magnit Tech**

Использовал:

- Webpack,
- SCSS,
- Swiper 8.

Для просмотра локально нужно:

Скачать репозиторий:

```bash
git clone https://github.com/ProjectOnGitHub/test-magnit.git
```

Установить зависимости:

```bash
npm install
```

Запустить проект:

```bash
npm run dev
```
